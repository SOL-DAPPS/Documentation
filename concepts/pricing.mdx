## Fee Breakdown

| Action | Fee |
|--------|-----|
| **Successful swap** (buy/sell) | 1% of transaction value |
| Failed transaction | Free |
| Read-only API calls | Free |
| Token creation | Free |
| Wallet operations | Free |
| Account management | Free |

<Note>
Fees are automatically deducted from the transaction. No invoices, no manual payments.
</Note>
